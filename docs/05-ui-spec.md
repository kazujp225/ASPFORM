# UI仕様

## 1. 画面一覧

### 管理画面
| パス | 画面名 | 説明 |
|------|--------|------|
| `/admin/login` | ログイン | ID/PW入力 |
| `/admin` | ダッシュボード | 検索・コピー |
| `/admin/groups` | グループ一覧 | CRUD |
| `/admin/groups/[id]` | グループ編集 | 詳細編集 |
| `/admin/plans` | プラン一覧 | CRUD |
| `/admin/plans/[id]` | プラン編集 | 本文・テンプレ |
| `/admin/submissions` | 履歴一覧 | ログ閲覧 |

### 顧客画面
| パス | 画面名 | 説明 |
|------|--------|------|
| `/p/[slug]` | 入力 | 顧客情報入力 |
| `/p/[slug]/confirm` | 確認・同意 | 条件確認 |
| `/p/[slug]/complete` | 完了 | mailto/コピー |
| `/error` | エラー | 無効リンク等 |

## 4. コンポーネント仕様

### トースト
- 位置: 右下固定
- 表示時間: 3秒
- メッセージ例: 「URLをコピーしました」

### Stepper（顧客画面）
- 4ステップ: 入力 → 確認 → 同意 → 完了
- 現在ステップ: 塗りつぶし
- 完了ステップ: チェックマーク

### コピーボタン
- アイコン: 📋 (clipboard)
- ホバー: 背景色変化
- クリック後: ✓ に1秒間変化
